FAQs

What is this site all about ?
Using this site, you can send gifts to your friends on Orkut. Select a gift image to be placed inside the gift wrapper and select the time when the gift is allowed to be opened. Then copy paste code at the bottom of the page to the orkut scrap book of your friends. Your friend can see the gift at the time you have specified

Can the flash code from this site be embedded in other sites ?
Yes, any sites that accepts flash can use this. You can check out MySpace, blogs etc. If you can send flash in e-mails, you can use this code for that too.

Sometimes gifts do not appear, why ?
The gift images are loaded from an external server and may take some time to be displayed. Check if the Gift URL that you have specified is valid. Currently you can only upload JPEGs. We are working on getting GIFs and PNGs. Images with white background will look good on the themes.

Why do you only have templates for christmas and birthday ?
We wanted to get the first cut out, so we went ahead with christmas and birthday. However, our "very talented" graphics designer is working hard to get skins / themes for other occassions also.

Are you guys planning to add more features ?
Sure, tons of more feature requests. Typically, things like letting only the reciever see the gifts (Deepika's idea), longer message lengths (Simi wanted this), customizable backgrounds (courtsey Jyothi) and stuff. If you have an idea, send it across to us.
